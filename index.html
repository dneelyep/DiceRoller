DiceRoller

This program is a simple dice roller, meant to be used to play tabletop games such as Dungeons and Dragons.

To use the program, run it and input the number of each type of dice you want to roll, then click "Roll!". Use the "Reset" button to change all dice numbers to 0. Enjoy!
